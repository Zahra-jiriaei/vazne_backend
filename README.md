# Vazne

## Overview

**Vazne** is a university team project developed for the **Systems Analysis and Design** course at the Iran University of Science and Technology. The project is an online sports store that allows users to browse sports products, manage their shopping experience, and access fitness-related services.

The project was developed by a four-member team following the **Scrum** framework, with dedicated **Scrum Master** and **Product Owner** roles. The team consisted of two frontend developers and two backend developers.

## Roles

Amirhossein Safaei Mehr and Zahra Jiriaei  worked as Backend Developers, using Python and Django to implement server-side functionalities, business logic, and database integration.

## Features

* User registration and authentication
* Sports coach directory
* Training courses catalog
* Digital wallet management
* Shop
* Product browsing
* Shopping cart

> **Note:** Online payment was intentionally excluded because this was an academic project.

## Tech Stack

* **Backend:** Python
* **Framework:** Django
* **Database:** SQLite
* **Development Methodology:** Scrum
* **Version Control:** Git
